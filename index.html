Drop Stack
You can drag and drop A,B and C into the folder and see new blocks appear in the drop stack. The blocks can then also be removed using the delete icon, which will also update the count bubble on the folder. This app is made without any framework, library or CDN except font-awesome which I used to get the dustbin icon in the stack blocks. Unfortunately, touch support could not be accomplished using plain vanillaJS despite the best of my efforts.
